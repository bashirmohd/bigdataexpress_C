Setting up DTN services
=======================

On each **DTN**, it is necessary to install and set up:

* BDE Data Transfer Engine, mdtmftp+ server (``mdtm-ftp-server`` binary).
* BDE DTN Agent, as ``bdeagent`` systemd service.
* Telegraph agent.
* X.509 security.

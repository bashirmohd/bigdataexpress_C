============================
How to access BDE web portal
============================

Once the web portal is up and running, it should be accessible at
https://example.net. Please replace ``example.net`` with the DNS name
of the node where you have set up the web portal.

.. image:: images/bde-portal-home.png

The portal has a bulit-in ``admin`` account with a default
password. Please contact the BDE development team for the password.

.. image:: images/bde-portal-admin-home.png
